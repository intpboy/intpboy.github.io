```cpp
// 산술연산자: +, -, *, /, %
int a = 7 + 3; // 합: 10
int a = 7 - 3; // 차: 4
int a = 7 * 3; // 곱: 21
int a = 7 / 3; // 몫: 2 
int a = 7 % 3; // 나머지: 1
int a = 10 + 5 * 3 // 25 
```

귀찮으니 각 연산자에 대한 설명은 생략함.

주의할 것은 / 는 피연산자가 둘 다 정수형이면 몫만 계산하고 소숫점은 버림(7/3 = 2.3333 이 아니라 2)

그러나 하나라도 실수형이면 정확한 계산이 이루어짐
