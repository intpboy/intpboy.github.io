# 사용자 입력(cin)

```cpp
    cout << "what is your name"<< endl;
    cout << "you: ";
    cin >> name ;
    cout << "hello ";
    cout << name << endl;
```

입/출력내용:

what is your name

you: ***intpboy***

hello intpboy

cin 함수는 iostream 내장 함수이다.

예시에서 string자료형의 name 변수를 초기화하지 않고 선언한 뒤

cin 함수를 통해 사용자가 직접 값을 입력하도록 했다.

또한 cin 함수로 입력되는 값이 name 변수에 저장되는 것이므로 cout과 반대로 >> 를 사용하는 것을 볼 수 있다.
